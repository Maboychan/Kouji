# TempGraph.py - 麹温度管理システム v1.0.0

麹造りにおける温度変化を美しく可視化し、記録するためのiPad専用スクリプトです。

## 🌟 主な機能
1. 温度データの自動記録：
   クリップボード経由、またはiOS共有シート（Share Sheet）からデータを取り込み。
2. グラフ自動生成：
   経過時間に応じた動的なグラフサイズ調整と、3層のレイヤーによる美しい折れ線描画。
3. インテリジェントなプレビュー：
   - 共有シートから実行：1秒間のクイックプレビュー（作業を邪魔しません）
   - エディタから実行：コンソールにグラフをしっかり表示
4. 履歴管理：
   「0️⃣」を含むコメントを入力することで、自動的に新しいデータファイル（.temp）を作成し、バッチを切り替えます。

## 📂 フォルダ構成
```
Documents/
 └ dev/
    └ Koji/                (任意の場所)
       ├ TempGraph.py      (本プログラム)
       └ background.png    (グラフの背景画像：必須)
       
This iPad                  (Pythonista3のルートディレクトリ)
 └ temp_data/              (自動生成：データ、ログ、画像が保存されます)
    ├ 20260313.temp        (温度データ)
    ├ 20260313_graph.png   (グラフ画像)
    ├ 20260313_haze_0.png  (破精廻り写真) 
    ├ 20260313_haze_1.png  (破精廻り透かし写真)
    └ log.txt              (ログ)
```

## 🚀 使い方
### A. 共有シートから（推奨）
1. 他のメモアプリなどで温度データ（例：2026-03-15 13:43 37.9）を選択。
2. 共有メニューから「Pythonista」を選択し、本スクリプトを実行。
3. グラフが1秒間プレビューされ、自動的にデータが保存・クリップボードにコピーされます。

### B. エディタから
1. 温度データをクリップボードにコピーします。
2. Pythonistaで本スクリプトを実行します。
3. 追加の確認ダイアログが出た後、コンソールにグラフが表示されます。

## 📊 データフォーマット
YYYY-MM-DD HH:MM 温度 [コメント]
（例）2026-03-15 13:43 37.9 ヒーターOFF

※コメントに「0️⃣」を入れると、その日の日付で新しい .temp ファイルが作成されます。
